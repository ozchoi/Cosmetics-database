# Open Data Licensing Notes

This project preserves provenance and attribution metadata for open-data imports, but final licensing decisions require legal review before public production use.

## Technical Support

- Preserve source attribution at record level.
- Preserve original external IDs.
- Preserve licence name and URL.
- Record source-derived fields separately from independently observed fields.
- Generate machine-readable source attribution metadata from import results.
- Include attribution and field provenance in export workflows.
- Maintain an import manifest for each reproducible job.
- Do not remove source attribution during deduplication or merging.

## Blocking Checklist Before Public Production Import

- Licence version reviewed.
- Attribution wording reviewed.
- Adapted-database obligation reviewed.
- Public database licence decision documented.
- Image licensing reviewed separately.
- Commercial-use implications reviewed.
- Redistribution and export obligations reviewed.

## Source Notes

Open Beauty Facts data is expected to involve Open Database Licence terms. The platform records ODbL-oriented attribution and field provenance, but this is not a legal conclusion.

PubChem and EPA CompTox are used for approved identity or structured staging workflows. Their records must not become consumer-facing risk claims without source review, relevance assessment, and reviewer approval.
