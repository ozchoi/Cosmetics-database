# Database Licensing

This project preserves source attribution and review status at record level, but no final legal conclusion is made here.

Before production publication of imported public data, review:

- licence version
- attribution wording
- adapted-database or share-alike obligations
- image licensing
- commercial-use implications
- redistribution and export obligations
- whether imported fields can be mixed with independently observed fields

EWG Skin Deep, CosDNA, and similar consumer websites are reference-only unless reuse rights are explicitly reviewed and approved.
