# Data Sources

Source hierarchy:

- Official and primary sources can support active claims when the claim context, route, concentration, population, and locator are preserved.
- EWG Skin Deep, CosDNA, and COSMILE are discovery or cross-check references only in this dataset.
- Open Beauty Facts remains isolated from this import until licensing review is complete.
- Brand pages may support product ingredient observations, but `package_photo_verified` remains false unless a real package photo is reviewed.

Public websites being readable does not imply permission to bulk copy, scrape, or republish their data.

Consumer-facing claims must link to a source record with publisher, title, category/type, jurisdiction, publication date or version where available, access date, exact locator, external URL, evidence relationship, evidence grade, and reuse/licence status where relevant.
